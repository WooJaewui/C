 # while, for, do ~ while.

while문.
while( 조건식 ) {
	// 조건식이 1( 참 )일 경우 반복할 문장 ;
	// 조건식이 0( 거짓 )일 경우 실행되지 않는다.
}
실행할 문장이 한 문장일 경우 { }를 생략할 수 있다.

for문.
for( i=0 ; i<10 ; i++ ) {
	// i<10이 1( 참 )일 경우 반복할 문장 ; 
	// 10번 실행된다.
}
( 초기식, 조건식, 증감식 ) 이라고 부른다
i<10이 참이면 { }안에 문장이 실행된 후에 i++이 된다.

do ~ while문.
do {
	// while문에 상관없이 1번이상 무조건 실행하고, while문 조건식이 1( 참 ) 일 경우 반복할 문장 ;
} while ( 조건식 ) ;
최소 1번이상 실행을 한다.



 # 반복문 활용.

중첩 반복문 예제.
별찍기, 구구단.

중첩 반복문의 break문.
가장 가까운 반복문 또는 switch문 하나만 종료한다.

조건식의 true, false.
false : 0.
true : 0이 아닌 모든 값.

continue.
반복문 안에서 continue를 만나면 반복문의 끝으로 이동한다.
for문이면 끝으로 이동해서 증감식 후 다시 조건을 확인한다.
