 # 함수의 작성과 사용.

표준 라이브러리.
자주 사용되는 함수들을 표준으로 미리 만들어 모아놓은 것.

함수 만들기.
타입 함수이름( 매개변수1, 매개변수2, ... ) {
	함수 호출시 실행할 코드 ;
}
'타입부터 )' 까지를 "함수 헤드( 함수 머리 )" 라고 한다.

함수 실행 순서.
main( )과 같은 startup 코드에서 작성된 순서대로 실행.
a함수가 호출되면 a함수의 내용이 실행되어 값을 리턴하고 a함수 뒤에서부터 순서대로 다시 코드가 실행.

함수 선언.
함수를 사용하기 위해 함수 헤드를 미리 적어주는 방법.
예 : 타입 함수이름( 매개변수1, 매개변수2, ... )


 # 여러가지 함수 유형.

매개변수가 없는 함수.
타입 함수이름( void ) { 
	함수 호출시 실행할 코드 ;
}

반환값이 없는 함수.
void 함수이름( 매개변수1, 매개변수2, .. ) { 
	함수 호출시 실행할 코드 ; ( return 생략가능 )
}
반환값이 없기 때문에 연산자와 함께 사용할 수 없다.

매개변수, 반환값이 없는 함수.
void 함수이름( void ) {
	함수 호출시 실행할 코드 ;
}

재귀함수.
타입 함수이름( 매개변수1, 매개변수2, ... ) { 
	함수이름( ) ;
	함수 호출시 실행할 코드 ;
}
반복문처럼 사용할 수 있다 - 증감식 등을 내부에 설정해서 필요한만큼 함수를 실행할 수 있다.
반복하다가 하나의 함수에 할당된 메모리 공간을 넘어섰을 때 재귀함수가 자동으로 종료된다.
함수 내부에 호출된 자기자신 함수는 새로운 함수이다.
