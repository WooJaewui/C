 # 배열의 선언과 사용.

배열 선언.
타입 배열이름[ 배열의 크기 ] ;

index.
배열의 위치를 의미하며 0 ~ ( 배열의 크기-1 ) 까지이다.

배열에 초기화.
타입 배열이름[ 배열의 크기 ] = { 값1, 값2, ... } 
( 배열의 크기 - 1 )개까지 값을 초기화할 수 있다.
( 배열의 크기 -1 )보다 작게 값을 초기화하면 나머지 값은 0으로 초기화된다.
초기화할 때 배열의 크기를 생략할 수 있다 - 값의 개수만큼 배열의 크기가 정해진다.

배열에 값 대입.
배열이름[ index ] = 입력값 ;

배열의 크기 함수.
sizeof( 배열이름 ) / sizeof( 배열이름[ 0 ] )  : 배열의 전체 byte를 배열의 하나의 요소로 byte로 나누면 배열의 크기를 알 수 있다.



 # 문자를 저장하는 배열.

null문자.
ASCII코드 0으로 모든 비트가 0인 값.
문자열에서 문자가 끝났음을 표시하기 위해 사용.

문자열 복사 함수.
# include <string.h> ( strcpy의 선언부가 저장되어 있다 )
strcpy( 문자열 변수이름, "복사할 문자" ) ;
예 : stycpy( str , "Hello world" ) ;

String 주의사항.
scanf( "%s" , str )을 통해 입력을 받을 때 스페이스, 탭, 엔터를 입력하면 문자열이 끝났다고 판단한다.
gets( str ), puts( str )을 통해 스페이스, 탭, 엔터를 입력해도 그대로 입력받는다. ( puts는 항상 줄바꿈을 한다 )
