 # 프로그래밍과 C언어.

프로그램.
일의 순서다.

프로그램 실행 순서.
저장공간 확보 -> 데이터 입력 -> 처리 -> 출력.

프로그래밍 언어.
컴퓨터가 프로그램을 실행시킬 수 있게 컴퓨터가 이해할 수 있는 언어로 프로그램을 만드는 것.

C언어의 역사.
1972, 벨연구소, 데니스 리치와 켐톰슨이 개발.
B언어에서 좀 더 개발되어 C언어가 되었다.

C언어의 특징.
하드웨어 제어, 호환성, 이식성 함수 사용.

C언어에 종류.
K&R C : 1978년 Kernighan과 Ritchie가 만든 C Programing Language.
ANSI C : American National Standard Institue 미국표준기구에서 정한 표준.
C99 : ISO International Standard Organization에서 99년에 정한 표준. ( 최근 )

Editor.
Visual C++



 # 컴파일러 사용법.

소스코드.
메모장, Editor 등으로 ASCII 코드로 작성한 text 파일.

ASCII 코드.
1바이트( 8비트 )로 데이터를 저장하는 방식.
타자로 칠 수 있는 모든 문자들은 ASCII 코드로 변환이 가능하도록 되어있다.
text 파일에 A를 작성하면 예를 들면 0100 0001와 같이 1바이트로 표현된다. ( A = 65 )

명령어.
CPU마다 +, -, A, 10, ... 와 같은 명령어들은 다르다.

컴파일.
코드화된 문자를 명령어들의 집합으로 바꾼다.

정리.
소스코드 작성 -> 컴파일 -> 운영체제에서 실행.
