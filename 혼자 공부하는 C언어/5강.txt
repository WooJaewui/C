 # if문.

제어문의 종류.
선택문 : if문, switch-case문.
반복문 : while문, for문, do-while문.
분기문 : break, continue, goto, return.

if문.
if ( 조건 ) { 
	// 조건이 1일 때 실행할 코드 ;
}
중괄호 스코프를 주의해서 사용해야 된다.
실행할 문장이 하나일 경우 중괄호를 생략할 수 있다.

if - else문.
if ( 조건 ) { 
	// 조건이 1( 참 )일 때 실행할 코드 ;
} else {
	// 조건이 0( 거짓 )일 때 실행할 코드 ;
}

if - else if문.
// 두 개 이상의 조건을 확인하기 위해 else if를 활용한다.
if ( 조건1 ) {
	// 조건1이 1( 참 )일 때 실행할 코드 ;
} else if ( 조건2 ) { 
	// 조건2가 1( 참 )일 때 실행할 코드 ;
} else {
	// 조건1과 조건2가 모두 0( 거짓 )일 때 실행할 코드 ;
}



 # if문 활용과 switch문 ~ case문.

중첩 if문.
if ( 조건1 ) { 
	if ( 조건2 ) {
		// 조건1과 조건2가 1( 참 )일 경우 실행할 문장 ;
	} else {
		// 조건1이 1( 참 )이고 조건2가 0( 거짓 )일 경우 실행할 문장 ;
	}
}
중첩 if문을 사용하면 일반 if문보다 불필요한 조건 검사를 줄일 수 있다.

중첩 if문 주의사항.
한 문장인 경우 { }를 생략할 수 있는데 이때 내부 if문과 외부 if문의 else를 명확하게 해줘야된다.

switch ~ case문.,
switch( 값 ) {
	case 조건1 : 
	// 값이 조건1일 경우 실행할 문장 ;
	break ;
	case 조건2 :
	// 값이 조건2일 경우 실행할 문장 ;	
	break ;
	...
	default :
	// 조건에 일치하는 값이 없는 경우 실행할 문장 ;
}
값에는 정수로 변형 가능한 모든 것이 들어갈 수 있다. ( 조건식, 함수, .. )
break를 생략하면 조건에 맞는 case를 모두 실행한다.
